HOW TO ADD A FORKLIFT

1- Go to Window/Package Manager and install PostProcessing package

2- Drag the forklift prefsb of the Prefabs folder, and the FPS if you want

3- Create a Canvas and 3 texts, , remember that you must to import the text mesh pro package, with the package manager

4- Asign the gear and speed text to the variables of the forklift script, the enter text too 

5- Drag your FPS controller to the "FPS" variable of the same script, (you can use the FPS of the Prefabs folder)


Also you have a demo scene in that you can see how everything works, and if you want, you can use that scene to build an entire game
